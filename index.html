Function of Babel

Check this function. It looks harmless enough: give it an x and a y, and it decides whether a tiny square should be on or off.

1 2 < ⌊ mod ( ⌊ y 17 ⌋ 2 - 17 ⌊ x ⌋ - mod ( ⌊ y ⌋ , 17 ) , 2 ) ⌋

Hidden inside it is every 106 × 17 pattern you can imagine; you only need to find the y-coordinate where it appears.
