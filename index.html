About this project and function

This whole thing follows Tupper’s self-referential formula: a suspiciously small equation that secretly contains every 106 × 17 pattern, if you know which y to ask for.

Function of Babel is a dummy project, an interactive playground so you can draw, scroll, and generally mess with the function instead of just staring at it on Wikipedia. It’s loosely inspired by libraryofbabel.info: same vibe of infinite shelves and finite rules, except here you scroll through y instead of hexagonal galleries.

It works on a phone, but you’ll have a much nicer time on a laptop or desktop: more room to draw, drag the chart (and pretend you know what you’re doing).

Want to see how the trick is wired up? Peek at the GitHub repo.
